# federated-hgnn-anomaly-detection
Privacy-Preserving Detection of Anomalous Identity Access Patterns in Multi-Cloud Environments using Federated HGNN

## Datasets

This project uses two publicly available datasets. 
Download them and place them in the paths below before running the notebook.

### Invictus-IR AWS CloudTrail Dataset
- Source: https://github.com/invictus-ir/aws_dataset
- Place JSON files in: `Dataset/Raw/CloudTrail/`
